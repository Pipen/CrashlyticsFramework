CrashlyticsFramework
====================

This contains the great Crashlytics Framework to be linked as a CocoaPod

## Usage

``` ruby
pod 'CrashlyticsFramework', :git => "https://github.com/bpoplauschi/CrashlyticsFramework.git"
```
